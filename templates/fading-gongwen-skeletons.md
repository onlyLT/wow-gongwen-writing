# 法定公文骨架模板（大纲加速起点）

用于工作流第 3 步"先出大纲"。占位 `【】` 处向用户取料，不杜撰。

## 通知（下行）
```
标题：【发文机关】关于【事由】的通知
主送：【各乡镇（街道）、县直有关单位】

  【依据/目的：根据…／为…】，现将【事项】通知如下。

  一、【事项一：内容+要求】
  二、【事项二：责任分工/标准】
  三、【工作要求：时限、落实】

请遵照执行。（或：特此通知。）

                              【发文机关（署名）】
                              【2026年×月×日】
（附注/联系人：如需）
```

## 请示（上行，一文一事，单主送，标签发人）
```
标题：【发文机关】关于【请求事项】的请示
主送：【一个上级机关】                    签发人：【×××】

  【请示缘由：背景、依据、必要性、问题——写充分，是说服核心】。

  为此，特请示如下：【明确、具体、可批复的请求事项】。

以上请示，请予批准。（或：当否，请批示。）

                              【发文机关（署名）】
                              【2026年×月×日】
（附注：联系人及电话）
```

## 报告（上行，陈述性，不夹带请示）
```
标题：【发文机关】关于【事项】的报告
主送：【上级机关】                        签发人：【×××】

  【概述/缘由】。现将有关情况报告如下。

  一、【工作进展与成效（数据）】
  二、【存在的问题】
  三、【下一步打算】

特此报告。（或：以上报告，请审阅。）

                              【发文机关（署名）】
                              【2026年×月×日】
```

## 批复（下行，答复请示）
```
标题：【发文机关】关于【事由】的批复
主送：【请示单位】

  你单位《关于【…】的请示》（【〔年〕号】）收悉。经研究，现批复如下：

  【同意/原则同意/不同意 + 具体意见、条件、要求】。

请遵照执行。（或：此复。）

                              【发文机关（署名）】
                              【2026年×月×日】
```

## 函（平行，不相隶属机关）
```
标题：【发文机关】关于【事由】的函
主送：【对方机关】

  【我单位因…／缘由】。现【商洽/询问/请予批准】如下：【事项】。

  请予支持并函复为盼。（复函：特此函复。）

                              【发文机关（署名）】
                              【2026年×月×日】
```

## 通报（下行，表彰/批评/情况）
```
标题：【发文机关】关于【表彰×××／×××问题】的通报
主送：【各乡镇（街道）、县直各单位】

  【事实：表彰/批评的事实经过、典型情况——写清楚再定性】。

  【评析/决定：定性 + 给予何种表彰/处理 + 原因剖析】。

  【要求：号召学习／引以为戒、整改要求（分条）】。

                              【发文机关（署名）】
                              【2026年×月×日】
```

## 意见（下行，实施/指导意见）
```
标题：【发文机关】关于【事由】的实施意见
主送：【各责任单位】

  【背景、依据、必要性】。经研究，提出如下意见。

  一、总体要求（指导思想、基本原则、目标）
  二、主要任务（分条，每条含内容+牵头/责任单位）
  三、保障措施（组织领导、经费、督查考核）

                              【发文机关（署名）】
                              【2026年×月×日】
```

## 会议纪要（第三人称，可不盖章）
```
标题：【会议名称】纪要

  【×年×月×日】，【主持人】主持召开【会议】。【出席/列席人员】。会议【听取了…研究了…】。现将议定事项纪要如下：

  一、【议定事项一】
  二、【议定事项二，含分工、时限】

                              【发文机关（署名）】
                              【2026年×月×日】
```
