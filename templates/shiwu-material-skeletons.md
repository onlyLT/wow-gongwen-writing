# 事务性材料骨架模板（大纲加速起点）

用于工作流第 3 步"先出大纲"。占位 `【】` 处向用户取料。

## 领导讲话稿（部署动员类）
```
标题：在【会议/场合】上的讲话
        —— 【领导职务 姓名】

  开头：今天召开【会议】，主要任务是【…】。刚才，【…作了部署/通报】。下面，我讲【三点】意见。

  一、提高认识，深刻把握【工作】的重要意义
     （讲形势、讲意义、统一思想）
  二、突出重点，扎实抓好【几项】关键任务   ← 主体，篇幅最大
     （一）【任务一】（二）【任务二】（三）【任务三】
  三、加强领导，确保各项任务落地见效
     （压实责任、强化保障、严格督查——常"三点要求"）

  结尾：号召鼓劲，【表决心/提希望】。
```

## 工作总结（年度/专项）
```
标题：【单位】【年度/专项】工作总结

  一、总体情况（一年/一项工作的背景与总评价）
  二、主要做法与成效           ← 主体
     （一）【板块一：做法+成效数据】
     （二）【板块二】（三）【板块三】
  三、存在的问题和不足（客观、具体，不可省）
  四、下一步工作打算（针对问题，务实举措）
```

## 工作方案 / 实施方案
```
标题：【事项】工作方案（实施方案）

  一、指导思想（依据、总体要求）
  二、工作目标（总体目标 + 可量化的具体目标）
  三、主要任务与分工           ← 主体
     （每条：内容 + 牵头单位 + 责任单位 + 完成时限；可附任务分解表）
  四、实施步骤（动员部署／组织实施／总结验收 + 时间节点）
  五、保障措施（组织领导、经费保障、宣传引导、督查考核）
```

## 汇报材料
```
标题：【单位】关于【事项】的工作汇报

  一、基本情况
  二、主要做法与成效（数据支撑，突出亮点）
  三、存在的困难和问题
  四、下一步打算
  （五、需上级支持解决的事项——具体：要政策/资金/项目）
```

## 调研报告
```
标题：关于【课题】的调研报告

  一、调研背景与方法
  二、现状与主要成效（一手数据、案例）
  三、存在的问题及原因分析（剖析到体制机制层面）
  四、对策建议           ← 主体，分条、可操作
```

## 述职报告（第一人称，实事求是）
```
标题：【年度/任期】述职报告
        —— 【职务 姓名】

  开头：履职总述【任职时间、分管领域、总体评价一两句】。

  一、主要工作与成效           ← 主体（"绩"为重点）
     （一）【工作一：干了什么+成效数据】
     （二）【工作二】（三）【工作三】
  二、廉洁自律情况（述职述廉时必写）
  三、存在的不足（真诚、具体，不避实就虚）
  四、下一步打算／表态

  （德、能、勤可融入各段，不必单列凑格式）
```

## 政务信息 / 简报（短、快、新，300–800字）
```
标题：【观点式标题，突出亮点，如"××县三举措推动…成效明显"】

  导语：【一句话：谁、做了什么、有何成效】。

  主体：【做法1】…【做法2】…【成效数据】…（一事一报，短小精悍）
```
