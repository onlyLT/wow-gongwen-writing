# 写作内功：标题·谋篇·造句·改稿

定位：**可迁移的生成法则 + 少量种子例句**，不是语料库。种子只作仿写参照，禁止原句照搬堆砌——一篇材料形象化表达 2～3 处、古语金句 1～2 处为限，过量即踩「空泛/陈套」（见下「语言十弊」）。

## 一、标题技法

法定公文标题守三要素（发文机关+事由+文种，见 format-gbt9704.md），**不玩花样**。以下技巧只用于：政务信息、简报、讲话稿、汇报、总结、调研报告的主标题与各类材料的小标题。

六条生成式（每条附种子，按主题仿写）：

1. **对仗式**——上下句字数相同、结构相似。种子：「锚定目标抓落实，聚焦重点求突破」「稳字当头夯基础，进字为要促发展」。
2. **数字式**——用数字概括抓手。种子：「三个强化」「四张清单」「'1+3+N'工作体系」。
3. **问题导向式**——直接点问题，用于调研/通报/整改。种子：「××工作存在的突出问题及对策建议」「当前××领域面临的三大挑战及应对思路」。
4. **形象化（喻体）式**——找准与工作内容对应的喻体：基础→「压舱石」、引领→「指挥棒」「风向标」、防线→「防护网」「高压线」、合力→「同心圆」「组合拳」。种子：「织密防护网，筑牢安全墙」「找准改革发展'公约数'」。比喻要自然贴切，不牵强。
5. **成效式**——突出价值，用于总结/汇报/信息。种子：「创新举措破难题，××工作迈上新台阶」「××县三举措推动××成效明显」。
6. **经验式**——提炼做法，用于经验交流/简报。种子：「坚持'三个强化'，推动××高质量发展」「创新四项机制，破解××难题」。

**小标题成组法则**：一组小标题用同一生成式、保持结构对称（如都是"动词+喻体"或都是"在'××化'××中××"），忌一组内风格混杂、字数参差。

## 二、谋篇骨架（三个通用逻辑，套任何主体段）

1. **是什么—为什么—怎么办**：部署类（意见、方案、讲话）。
2. **情况—问题—原因—对策**：分析类（调研、汇报、总结）。
3. **总—分—总**：先亮观点，分条支撑，收尾落点。

**分条法则**：每个分条 = 一句提领句（动宾或判断句起头）+ 事实/数据支撑 + 一句落点（谁、何时、达到什么标准）。虚实配比：一句务虚（意义/站位）必须带一句务实（具体动作），只虚不实即空转。

## 三、造句与「翻译」法则

把口语要点、领导只言片语、专业材料转成公文语，本质是**三层翻译**：
1. 领导的思想 → 文字（忠实还原，不自由发挥）；
2. 零散的想法 → 结构（用上面三个骨架把碎片串起来）；
3. 专业语言 → 平实语言（让非专业读者能懂）。

**空话识别两标准**（命中即删）：①删掉这句话意思不变→空话，删；②这句话放任何文章都适用→套话，换成本单位实际。替换方向：空话→具体数据/事例；套话→本地做法；废话→直接进下一句。

**写短四步（先长后删）**：①不设限列出全部想说的 → ②分类合并同类项，每个观点只说一遍 → ③逐句压缩，删一切可删字词 → ④通读再删一轮。

**口语 → 公文替换表（种子）**：

| 口语/初稿 | 公文语 |
|---|---|
| 我觉得这个问题挺严重的 | 这一问题不容忽视 |
| 尽快落实、抓紧办 | 于×月×日前完成 |
| 相关部门要配合 | 由×局牵头，×单位配合 |
| 差不多完成了 | 已完成×%，达到序时进度 |
| 开了很多会研究 | （过程性表述，删或一句带过） |

## 四、语言十弊（成稿自检，配合 negative-checklist 第五类）

| 弊病 | 典型表现 | 改法 |
|---|---|---|
| 冗繁 | 一句话说三句 | 删一切可删字词 |
| 空泛 | 道理放之四海皆准 | 用本地数据替换通用表述 |
| 虚假 | 数据未核、案例道听途说 | 逐条核来源（P0） |
| 干瘪 | 满篇"加强/完善/推进"无内容 | 替换为具体措施 |
| 陈套 | 换个年份又一篇 | 写出今年的特点 |
| 模式化 | "在……正确领导下"开头 | 开门见山切入正题 |
| 模糊 | "相关部门""尽快""适当" | 明确主体、时限、标准 |
| 断路 | 上下句无逻辑连接 | 补段落过渡 |
| 脱节 | 开头结尾与主体两张皮 | 查全文是否围绕一个主题 |
| 冷漠 | 读完不知起草者态度 | 该有判断处敢下判断 |

## 五、素材种子库（每类 3～5 条，仿写不粘贴）

- **过渡（成绩→问题）**：「肯定成绩的同时，也要清醒看到，……还存在一些不容忽视的问题」「对照××要求和群众期盼，还有差距，主要表现在……」
- **过渡（问题→举措）**：「针对上述问题，将采取以下措施，切实加以解决」「下一步，重点抓好以下几方面工作」
- **数据句式**：「截至×月底，××已完成×%，超序时进度×个百分点」「今年以来共实施××项目×个，总投资×亿元，同比增长×%」
- **部署/落实**：「一分部署，九分落实」「久久为功、善作善成」「压实责任链条，层层传导压力，形成闭环管理」
- **收尾**：「路虽远，行则将至；事虽难，做则必成」「征程万里风正劲，重任千钧再出发」
- **古语（用前核对原文与语境）**：「道虽迩，不行不至；事虽小，不为不成」（强调落实）、「居安思危，思则有备，有备无患」（风险防范）、「知屋漏者在宇下，知政失者在草野」（联系群众）
- **高频四字词（按需取，忌连串堆砌）**：成效类——稳中有进、成效显著、亮点纷呈；部署类——突出重点、狠抓落实、落细落小、标本兼治；问题类——不容忽视、亟待解决、存在短板、举一反三；要求类——提高站位、强化担当、真抓实干、力戒空谈

## 六、改稿七步与优先级（「改稿/润色」分支的方法底座）

用户拿已有稿来改时，不走取料-大纲流程，按此执行：

**第一遍·诊断（七步顺查）**：①查文种（选对没有、标题三要素）→ ②查主送（主送/抄送对不对）→ ③查逻辑（框架清晰、层次分明）→ ④查事实（数据、政策依据、引文）→ ⑤查语言（对照上表十弊）→ ⑥查格式（对照 format-gbt9704.md）→ ⑦查错字标点。

**诊断输出按优先级分级**：

| 级别 | 内容 | 要求 |
|---|---|---|
| P0 | 事实/数据/政策表述、文种与行文方向 | 必须绝对准确，先改 |
| P1 | 观点/结论/定性表述、结构逻辑 | 经得起推敲 |
| P2 | 语言/格式/标点 | 规范工整 |

**第二遍·修改**：经用户确认诊断后，按 P0→P1→P2 逐级改；逐段润色时先开头（切题不超过 3 句）、再主体（删冗余、调语序）、后结尾与标题。

**第三遍·通读**：流畅性、前后口径一致性；重点复核数据、人名、地名、日期、引用文件（P0 项二次确认）。最后照常输出 negative-checklist 校验清单。
