# 文风与用语（政府口径）

政府口径 = **准确、平实、务实、简明**。重政策落地与数据事实，少空泛抒情。基层稿接地气、务实；市省厅稿规格更高、更凝练。三条总纲：**话要说准、事要说清、劲要使实**。

## 一、政府口径的基本文风

- **准确**：定性、数据、依据、表述不含糊。引文件写全称加文号，引数据可核。
- **平实**：陈述句为主，少形容词堆砌，少感叹。能一句说清不用三句。
- **务实**：举措落到"谁、干什么、何时完成、达到什么标准"，不停在口号。
- **简明**：一层一意，分条列项。段落不宜过长，避免"的的不休"。
- **得体**：上行文谦敬，下行文明确，平行文商洽；分寸与行文方向匹配。

## 二、结构套路（可直接化用）

### 开头引语（发文缘由）
- 依据式："根据《…》（文号）精神，" / "按照×××部署要求，"
- 目的式："为深入贯彻…，进一步…，" / "为切实做好…工作，"
- 背景式："当前，…形势/工作…。" / "近年来，我县…取得成效，同时…"
- 承接式（引出事项）："现将有关事项通知如下：" / "现就…提出如下意见：" / "特请示如下："

### 过渡与提领
- 分点提领："一是…。二是…。三是…。" / "第一，…第二，…"
- 强调递进："尤其要…" / "特别是…" / "在此基础上，进一步…"
- 转折谈问题："同时也要清醒看到，…还存在…" / "但对照…要求，还有差距，主要表现在…"

### 结尾语（按文种，务必与文种匹配）
- 通知："请遵照执行。" / "请认真抓好落实。" / "特此通知。"
- 请示："以上请示，请予批准。" / "当否，请批示。" / "妥否，请批复。"
- 报告："特此报告。" / "以上报告，请审阅。"
- 批复："此复。" / "请遵照执行。"
- 函（去/复）："专此函达，请予复函。" / "特此函复。"
- 意见（上行）："以上意见，请审示。"
- 讲话稿："让我们…（号召），（表决心/祝愿）！"

## 三、称谓与规范用语

- 自称：机关用"我县/我局/我办/本机关"，不用"咱""我们单位"（口语汇报可"我们"）。
- 对上级："请上级/请省厅/请市政府"，敬语"审示、批示、批复、审阅、审定"。
- 对下级："各乡镇（街道）、县直各单位"，要求语"遵照执行、抓好落实、认真办理"。
- 规范简称：首次出现用全称并可括注简称（"武义县发展和改革局（以下简称县发改局）"），后文用简称统一。
- 时间表述：用具体日期或"截至6月底"，慎用"目前/近期"泛指；跨年用全称年份。

## 四、数字与计量书写

- **统计数据、行文字号、日期、百分比用阿拉伯数字**：`完成投资3.2亿元`、`同比增长12.5%`、`〔2026〕5号`、`2026年7月2日`。
- **成词、概数、缩略语用汉字**：`十三届四中全会`、`三大攻坚战`、`党的二十大`、`五六个`、`第一书记`。
- 大数值可"万/亿"为单位，前后统一（不要一处"3200万"一处"0.32亿"混用）。
- 百分数写"百分之几"或"%"，全文统一；范围用"至"或"—"（`12%至15%`）。
- 序数、条款序号见格式文件（`一、（一）1.（1）`）。
- 本节为数字书写**总规则**；发文字号、成文日期作为版式要素的写法细则以 `format-gbt9704.md` 二、四节为准，两处如有出入以彼为准。

## 五、政治性与规范表述（政府口径也要守）

- 引用领导重要论述、党中央决策部署、上级文件表述，须**准确、完整、不断章取义**，拿不准就留 `【待核对原文】`（占位符约定见 SKILL.md）。
- 涉及重大方针、政治定性、对外口径的表述，本 skill 不擅自发挥，提示由核稿人/宣传或保密部门把关。
- 机构、职务、地名、人名用规范全称，排名先后按惯例（党政领导、单位排序不可随意）。

## 六、避免的文风问题（详见 negative-checklist）

- 空话套话堆砌、"高度重视/切实加强"连用而无实招。
- 生造词、网络用语、口水话进正式公文。
- 长句缠绕、一段到底、逻辑跳跃。
- 只有原则、口号，不落任务分工与时限。
